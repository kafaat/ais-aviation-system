## ملاحظات - Notes

- جميع الأسعار بالريال السعودي (SAR)
- الأسعار مخزنة في قاعدة البيانات بالهللات (100 هللة = 1 ريال)
- جميع الرحلات بحالة "scheduled" (مجدولة)
- مواعيد الرحلات نسبية ليوم الغد من تاريخ تشغيل السكريبت
- لتشغيل بيانات البذر: `node scripts/seed-data.mjs`

## اختبار نظام الحجز - Booking System Test Flow

1. **البحث عن رحلة**: اختر مطار المغادرة والوصول والتاريخ
2. **اختيار الرحلة**: اختر درجة السفر (اقتصادي/أعمال)
3. **إدخال بيانات المسافرين**: الاسم، جواز السفر، إلخ
4. **الدفع**: يتم التوجيه إلى Stripe Checkout
5. **تأكيد الحجز**: بعد الدفع يتم تأكيد الحجز وإصدار التذكرة الإلكترونية
